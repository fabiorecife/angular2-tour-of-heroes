# Angular 2 - Tour of heroes

based on https://angular.io/docs/ts/latest/tutorial/ - 2016-02-18

## requirements

1. npm
2. typescript (npm i -g typescript)

## steps

```shell
git clone https://github.com/fabiorecife/angular2-tour-of-heroes.git
cd angular2-tour-of-heroes
npm install
npm start
```
